# Kitap Kulübü Yönetim Sistemi - Web Programlama Proje Ödevi

Bu proje, Web Programlama dersi için geliştirilmiş bir Fullstack Web Uygulamasıdır. 
Projede Node.js (Express) backend mimarisi ve mysql2 kütüphanesi standartlarına uygun ilişkisel veritabanı yapısı kullanılmıştır.

Proje Kurulumu ve Çalıştırma

Projenin yerel bilgisayarda çalıştırılması için aşağıdaki adımları takip ediniz:

 Proje kaynak kodlarını bilgisayarınıza indirin. VS Code terminalinde projenin kök dizinine gelin.
 Gerekli tüm bağımlılık paketlerini indirmek için şu komutu çalıştırın:
   npm install
   projeyi çalıştırmak için vs code terminalinde:
   node sunucu/baslatici.js
   yazın ve sistem aktif hale gelince tarayıcınızın adres çubuğuna   http://localhost:3000/index.html  yazın proje açılacaktır
